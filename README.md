# Sistema de Control Presupuestal de Dircetur-Tacna
el objetivo es a partir de la asignación de un monto gestionar el seguimiento de como se desarrolla el gasto de cada una de las partidas asignadas y generar reportes que permitan llevar el control y así poder ayudar en la toma de decisiones.



Howto contribute
----------------

1. Fork this repository.

2. Clone the presupuesto repository.

        git clone git@github.com:yourname/presupuesto.git
        cd presupuesto

3. Checkup the development branch.

        git checkout development

4. Init git flow accepting default values.

        git flow init

5. Make some changes in a new feature branch and commit them.

        git flow feature start {feature-name}
        git add {something}
        git commit
        git push

6. When you finish your changes close your feature

        git flow feature finish {feature-name}
        git push

7. Create a pull request in github.
